Q:UIButtonжғіиҰҒжү©еӨ§е®ғзҡ„е“Қеә”еҢәеҹҹ?
A:йҮҚиҪҪUIButtonзҡ„hitTestжҲ–pointInside.
йҮҚиҪҪhitTest:

жҲ–йҮҚиҪҪpointInside:

иҝҷйҮҢжҺЁиҚҗйҮҚеҶҷpointInsideж–№жі•.еӣ дёәйҮҚеҶҷhitTestж–№жі•йңҖиҰҒжіЁж„ҸеҲӨж–ӯviewзҡ„дёҖдәӣеұһжҖ§:if (self.userInteractionEnabled == NO || self.hidden == YES || self.alpha <= 0.01) return nil;еҰӮжһңдёҚеҲӨж–ӯдјҡеҮәBug,жҜ”еҰӮи®ҫзҪ®жҢүй’®hiddenдёәYESдәҶ,дҪҶжҳҜж‘ёеҲ°жҢүй’®зҡ„еҢәеҹҹ,жҢүй’®зҡ„е“Қеә”ж–№жі•дёҖж ·дјҡиў«и°ғз”Ё.иҖҢйҮҚеҶҷpointInsideж–№жі•еҲҷдёҚйңҖиҰҒиҖғиҷ‘иҝҷдәӣ.

PS:еҰӮжһңи®ҫзҪ®дәҶuserInteractionEnabled=NO;еҚідҪҝйҮҚеҶҷhitTestиҝ”еӣһжҢүй’®иҮӘе·ұ,жҢүй’®зҡ„е“Қеә”ж–№жі•д№ҹдёҚдјҡеӣһи°ғ.еә”иҜҘжҳҜзі»з»ҹеңЁеӣһи°ғд№ӢеүҚеҸҲеҲӨж–ӯдәҶдёҖдёӢиҜҘеұһжҖ§зҡ„еҖј.

userInteractionEnabledзҡ„иҜҙжҳҺеҰӮдёӢ:

When set to NO, touch, press, keyboard, and focus events intended for the view are ignored and removed from the event queue. When set to YES, events are delivered to the view normally. The default value of this property is YES.

During an animation, user interactions are temporarily disabled for all views involved in the animation, regardless of the value in this property. You can disable this behavior by specifying the UIViewAnimationOptionAllowUserInteraction option when configuring the animation.

еҪ“btnдҪңдёәз¬¬дёҖе“Қеә”иҖ…иў«иҝ”еӣһж—¶,еҰӮжһңuserInteractionEnabledеұһжҖ§дёәNO,еҲҷдёҺе®ғзӣёе…ізҡ„дәӢд»¶(и§Ұж‘ё,жҢүеҺӢ)йғҪе°Ҷиў«еҝҪз•Ҙ.еӣ жӯӨbtnзҡ„е“Қеә”ж–№жі•дёҚдјҡиў«и°ғз”Ё.

Q:зӮ№еҮ»еӯҗи§Ҷеӣҫи¶…еҮәзҲ¶и§Ҷеӣҫзҡ„еҢәеҹҹд»Қи®©еӯҗи§Ҷеӣҫе“Қеә”дәӢд»¶?
A:еҝ…йЎ»йҮҚиҪҪзҲ¶и§Ҷеӣҫ(**зҲ¶и§ҶеӣҫзҲ¶и§ҶеӣҫзҲ¶и§Ҷеӣҫ**дёүйҒҚ!!!)зҡ„pointInsideж–№жі•.иҝҷжҳҜз”ұдәҺеҰӮжһңзҲ¶и§Ҷеӣҫзҡ„pointInsideиҝ”еӣһNOзҡ„иҜқ,е…¶дёҠзҡ„еӯҗи§Ҷеӣҫе°ұдёҚдјҡиҝӣиЎҢhitTest,еӣ жӯӨйҮҚиҪҪеӯҗи§Ҷеӣҫзҡ„зӣёе…іж–№жі•е°Ҷж— ж•Ҳ.

еңЁйЎ№зӣ®дёӯйҒҮеҲ°дёҖдёӘеңәжҷҜ:collectionViewCellдёҠзҡ„жҢүй’®и¶…еҮәдәҶcellзҡ„иҫ№з•Ң,жғізӮ№еҮ»жҢүй’®и¶…еҮәйғЁеҲҶжҢүй’®д№ҹиғҪеӨҹе“Қеә”дәӢд»¶,жҢүз…§дёҠиҝ°еҠһжі•йҮҚеҶҷдәҶcollectionViewCellзҡ„pointInsideж–№жі•,дҪҶжҳҜеҚҙжІЎж•Ҳжһң.дәҺжҳҜеҸҲйҮҚеҶҷдәҶcellзҡ„hitTestж–№жі•,еҸ‘зҺ°hitTestж–№жі•иҝ”еӣһзҡ„жҳҜcellиҮӘиә«,е№¶жІЎжңүиҝ”еӣһbutton.иҝҷдёӢзіҠж¶ӮдәҶ,жҳҜдёҠйқўзҡ„зҗҶи®әй”ҷдәҶеҗ—?дёҠйқўзҡ„зҗҶи®әжҳҜжІЎжңүй”ҷзҡ„,еҲҶжһҗдәҶеҚҠеӨ©еҸ‘зҺ°ж №жң¬еҺҹеӣ жҳҜcellе№¶дёҚжҳҜbuttonзҡ„зҲ¶и§Ҷеӣҫ,buttonзҡ„зҲ¶и§ҶеӣҫжҳҜcontentView,иҖҢcontentViewзҡ„зҲ¶и§ҶеӣҫжүҚжҳҜcell.иҷҪз„¶cellзҡ„pointInsideиҝ”еӣһдәҶYES,дҪҶжҳҜcontentViewзҡ„pointInsideдҫқз„¶иҝ”еӣһзҡ„жҳҜNO.жүҖд»ҘдёҚдјҡеҶҚhitTest contentViewдёҠзҡ„еӯҗи§Ҷеӣҫ,buttonиҮӘз„¶е°ұжІЎжңүеҸҚеә”дәҶ.зңҹзӣёз»ҲдәҺжҳҺдәҶ,дҪҶй—®йўҳиҝҳжІЎи§ЈеҶі,еҰӮдҪ•и§ЈеҶіе‘ў?

з”ұдәҺcontentViewжҳҜcellзҡ„еҶ…йғЁеұһжҖ§жІЎеҠһжі•иҮӘе®ҡд№ү,жүҖд»ҘиҝҷйҮҢжҲ‘д»¬йҮҚеҶҷcellзҡ„hitTestж–№жі•,еҲӨж–ӯеҰӮжһңзӮ№еҮ»зҡ„зӮ№дҪҚдәҺжҢүй’®еҢәеҹҹеҶ…,еҲҷзӣҙжҺҘе°ҶжҢүй’®иҝ”еӣһ.иҝҷдёӢзӮ№еҮ»и¶…еҮәеҢәеҹҹжҢүй’®д№ҹиғҪеӨҹе“Қеә”дәӢд»¶дәҶ,е®ҢзҫҺ.
